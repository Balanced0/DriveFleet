# DriveFleet — Modern Car Rental Platform

<div align="center">

### Seamless Car Booking Experience for Modern Travelers

Explore, book, and manage rental cars with a clean and responsive full-stack web application.

</div>

---

## Features

- Secure Authentication with Email/Password & Google Login
- JWT Authentication with HTTPOnly Cookies
- Explore Cars with Search & Filter Functionality
- Real-Time Booking System
- Full CRUD Operations for Car Listings
- Personalized User Dashboard
- Fully Responsive Design
- Toast Notifications & Loading Spinners
- Protected Private Routes & APIs
- Modern and Clean UI Design

---

## Home Page
